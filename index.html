🔥 Новые добавленные команды

  • Modify + Stack Shells Maya автоматически наложит одинаковые шеллы друг на друга (нужно выделить одинаковые шелы)
  • Arrange + orient shells Ориентация шеллов
  • Edit mesh + add divisions Добавление подразделений(subdivision)
  • Edit mesh + circularize Скругление геометрии(loop tools)
  • Edit mesh + merge to center Схлопывание вершин к центру
  • Mesh tools + multi-cut (insert edge loop – ровные ребра) - нож для создания ребер (ctrl+ЛКМ)
  • B Soft Selection - пропорциональное выделение (Чтобы изменить радиус влияния, зажми клавишу B и удерживай ЛКМ (левая кнопка мыши), потом двигай мышь влево/вправо.)

Горячие клавиши

  • Все горячие клавишы работают только в Английской раскладке

Движение во вьюпорте

  • F приближение к выделенному обьекту
  • A отдолиться от камеры
  • Alt + ЛКМ Движение (ПКМ, Колесико)
  • Space + ПКМ меняем вид (ортогональный/обьектный)

Слои

  • P выделения детей, а потом родителя, делает слой главным
  • Shift + P деление детей
  • Ctrl + G создание группы для выделенных обьектов
  • Выбор одного компонента и стрелка вверх выделяет все элементы группы

Инструменты

  • Q инструмент выбора
  • W инструмент передвижения
  • E инструмент поворот
  • R инструмент масштабирования
  • Ctrl + D дублировать обьект
  • Shift + ЛКМ клонирование обьекта
  • Shift + D повтор последнего движения
  • J шаг на 5
  • Shift + ПКМ добавление обьектов

Работа в режиме Компонентный (режим редактора)

  • F8 переключения в обьектный и компонентный режим
  • F9 выбор вершин
  • F10 выбор ребер
  • F11 выбор граней
  • Ctrl + E екструдирование
  • Ctrl + B bevel/фаска скругление краев обьекта
  • Mesh + Fill hole сшивает 2 обьекта друг к другу
  • Mesh + mirror отзеркаливание
  • Edit mesh + Merge to center сшивает вертехсы в центре
  • Edit mesh + Extract(Detach) разделяет обьект на 2
  • Edit mesh + Bridge Сшивание обьекта полигонами
  • Mesh display + reverse Меняет внешнюю сторону на внутренную
  • Mesh display + harden harden edge (soften edge, soften/harden) – сглаживание или заострение
  • Ctrl + del удаление ребер и вершин из меша
  • Mesh tools + multi-cut (insert edge loop – ровные ребра) - нож для создания ребер (ctrl+ЛКМ)
  • Mesh tools + target weld сшивает вершины друг к другу
  • Modify + Freeze Freeze Transformations – сброс трансформации обьекта
  • 4 отображение сетки обьекта
  • 5 отображение обьекта
  • Select + Contiguous edges Если не выделяются ребра по контуру
  • Create + lights все имеющиеся света
  • Create + Cameras все камеры
  • ПКМ (удерж.) меню для выбора компонента
  • 2ЛКМ выделение (вершин, ребра и граней) по контуру до определенной точки с зажатым shift
  • Ctrl + 1 показывает только выделенный обьект
  • Ctrl + H скрыть выделенный обьект
  • Edit mesh + add divisions Добавление подразделений
  • Edit mesh + circularize Скругление геометрии
  • Edit mesh + collapse Схлопывание (коллапс) элементов

Центр обьекта pivot

  • Insert (D-удержание) – движение центра обьекта
  • V привязка центра к вершинам других обьектов
  • X привязка к сетке(grid)
  • D + V перемещение pivot с привязкой к вершинам
  • Возвращение pivot к центру обьекта выполняется Modify – Center pivot (рекомендуется добавить в custom)
  • Изменение чувствительности гизмы клавиша (+/-) (стрелки X-Y-Z)
  • Alt + shift + D отчистка истории(edit + delete by type + history)
  • ПКМ + assign assign existing matireal+lambert – Присваеваем дефолтный материал обратно
  • Ctr + Shift + ЛКМ Добавления инструмента в блок быстрых(постоянно используемых) инструментов в custom
  • Tool settings move snap settings + retaine component spacing – вставляем правильную привязку pivot

Анимация

  • Alt + V воспроизведение анимации
  • Alt + (</>) передвижение по кадрам (вперед/назад)
  • K + ЛКМ передвижение по кадрам
  • Создание ключевых кадров, а так же удаление
  • ПКМ + Key Selected (channels-свойство) добавление ключевых кадров
  • S добавление ключевых кадров для всех свойств обьекта
  • Shift + (W-E-R) добавление ключевых кадров по (перемещению-поворота-увеличению)
  • Shift + ЛКМ (на выбранном кадре) дает возможность переместить кадр
  • ПКМ + Delete Selected (channels-свойство) удаление кадров
  • Windows + animation animation editors + graph editor – графически редактор анимации
